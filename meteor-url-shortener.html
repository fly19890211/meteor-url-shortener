Enter in the original URL, and then a keyword or multiple keywords to define the short URL. The short url must not contain any spaces or special characters (can contain dashes, underscores, periods). The shortened URL will be http://go.meteor.com/m/{your_short_url_keyword}. Only users that created a URL will have the ability to delete it. However, one can over write a URL by just submitting another entry with the same short URL.
